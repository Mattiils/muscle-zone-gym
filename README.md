# Muscle Zone Gym Website

## Overview
Muscle Zone Gym is a responsive fitness website designed to promote a modern gym business online. The website allows users to explore membership plans, meet trainers, and contact the gym easily on desktop, tablet, and mobile devices.

The project was created using HTML5, CSS3 and JavaScript.

---

## Aims and Objectives

- Promote Muscle Zone Gym professionally online
- Provide clear membership pricing information
- Introduce available trainers and services
- Allow users to contact the gym through an interactive form
- Deliver a responsive and accessible user experience
- Create a modern fitness-themed website

---

## User Stories

### First Time Visitor
- I want to quickly understand what the gym offers.
- I want to view pricing before joining.
- I want to know if the gym looks professional.

### Returning Visitor
- I want to view trainers and services.
- I want to compare membership plans.

### Existing Member
- I want to contact the gym quickly.
- I want to access pages easily through navigation.

## Development Process

The website was developed in Visual Studio Code using separate HTML pages linked through a navigation menu.

### Key Design Decisions

- Dark background chosen to suit gym branding
- Blue accent colour used for energy and consistency
- Card layouts used for membership plans
- Grid layouts used for responsiveness
- Simple navigation used for usability

### Challenges Faced

- Ensuring all pages linked correctly
- Making design responsive on mobile
- Fixing background image path
- Organising project folders correctly

### Solutions

- Used reusable navigation code
- Used CSS Grid and media queries
- Tested paths carefully
- Structured files into css / js / images folders
